Description

Interface to the \(\texttt{LORENE}\) method of class \(\texttt{Bin_NS}\) with the same name, that returns 1 if the energy density is nonpositive, or if the specific energy is nonpositive, or if the pressure is nonpositive, at the specified point; it returns 0 otherwise
