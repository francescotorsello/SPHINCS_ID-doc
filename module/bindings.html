Define SUBROUTINES that create and destroy * a \(\texttt{Et_diffrot}\) object from \(\texttt{LORENE}\), * and call its member methods * that extracts the \(\texttt{LORENE}\) initial data * from the spectral data saved in the 'resu' * binary files produced by \(\texttt{LORENE}\) * * For more information on how to link C functions to Fortran, see here. * FT 24.10.2021 * *
