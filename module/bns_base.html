bnsbase

This MODULE contains the definition of TYPE bnsbase, which is an ABSTRACT TYPE representing any possible type of binary neutron star (BNS) initial data (ID) to be set up for \(\texttt{SPHINCS_BSSN}\). That is, BNS ID produced with \(\texttt{LORENE}\), with \(\texttt{FUKA}\), etc.

PROCEDURES and variables shared by all the types of BNS ID should belong to TYPE bnsbase, as they are inherited by its EXTENDED TYPES that represent more specific types of BNS ID.

type, public, ABSTRACT, extends(idbase) :: bnsbase

ABSTRACT Base TYPE for BNS ID for \(\texttt{SPHINCS_BSSN}\) (produced with \(\texttt{LORENE}\), or with \(\texttt{FUKA}\), etc.; or produced with the same tool, but read in different ways, for example by linking to the LORENE library, or reading the ID from a lattice, etc.)
