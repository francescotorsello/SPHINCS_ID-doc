module procedure compute_and_export_bssn_constraints_grid module module subroutine compute_and_export_bssn_constraints_grid(THIS, id, namefile, name_logfile)

Compute, store, analyze and print the BSSN constraints to a formatted file. The computation is done by importing the LORENE hydro ID on the gravity grid, without any information on the particles.
